`rotl`, an R package to access the Open Tree of Life data

francois

Hello all,

In collaboration with @josephwb and David Winter, we recently released an R package that can be used to query and download data stored in Open Tree of Life. In particular, it is possible to directly access subsets of the synthetic tree, and to trees from individual studies that contribute to the synthetic tree. The package is available from CRAN and from GitHub.

We also published a pre-print describing this package and would appreciate feedback on the manuscript and the package.

Thank you,
